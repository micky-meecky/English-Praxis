## 💡 关于 English Praxis

**English Praxis**（原项目代号 `IELTS Master`）是一款专为深度英语阅读和标准化考试备考而生的高阶 AI 辅助软件。我们厌倦了传统、割裂的查词与做题体验，致力于打造一个**沉浸式、一体化、数据驱动**的本地优先阅读工作流。

通过将最前沿的大语言模型（LLM）能力深度无缝融合到阅读场景中，English Praxis 提供全链路的学习支持——从文章结构剖析、长难句降维打击，到全仿真题型生成与错题复盘，帮助用户以最高效的路径达成阅读能力的本质跃迁。

## ✨ 核心亮点 (Features)

🚀 **原生沉浸阅读体验**
- **无感查词下钻**：一点即刻释义，支持从基础词义到语境"熟词僻义"的无限嵌套深度剖析。
- **长难句终结者**：一键解构复杂句型，AI 智能抽离主谓宾骨架，层层解析修饰成分。
- **本地优先架构**：你的所有生词本、高亮批注与错题记录均加密存储于**本地硬盘**，绝对的隐私安全，且响应极速。

🧠 **全能 AI Copilot 伴学**
- **多模型支持**：自带主流大模型接口（DeepSeek, GPT-4, Claude，Kimi 等），只需填入你自己的 API Key 即可开启满血 AI 体验，成本透明可控。
- **上下文感知对话**：侧边栏 AI 助手时刻感知你当前阅读的段落，随时提问，精准解答逻辑疑点。

🎯 **多考试专项模式**
- **高仿真题库生成**：AI 针对不同词汇水平动态生成雅思、托福、考研阅读级文章。
- **全题型支持**：覆盖 T/F/NG、Heading 匹配、Summary 填空、完形填空等多维度实战题型。
- **错题高维解析**：不仅仅是给出答案，AI 会像名师一样为你一对一复盘干扰项陷阱与底层思维逻辑。

📊 **闭环学习数据网络**
- **能力追踪体系**：构建个人的生词遗忘曲线与薄弱题型雷达图。
- **引经据典背单词**：所有在文章中查询过的单词会自动关联上下文场景，在背词时为你提供最真实的回忆线索。

目前处于 Beta 阶段，请通过以下渠道获取最新桌面客户端（当前优先支持 Windows 10/11）：

- **国内极速下载**：
  - [夸克网盘](https://pan.quark.cn/s/REPLACE_WITH_YOUR_LINK)
  - [百度网盘](https://pan.baidu.com/s/REPLACE_WITH_YOUR_LINK) (提取码: `REPLACE`)
- **GitHub Releases** (官方更新源)：
  - [前往 Releases 页面查看所有版本](https://github.com/micky-meecky/English-Praxis/releases)

**安装步骤**：
1. 下载最新版 `.exe` 安装包。
2. 按照系统提示完成安装。
3. 首次启动需获取授权，导入个人授权码，即可解锁全部功能。
4. 随后请进入「设置」，配置所需的第三方 AI 服务商 API Key，即可开启满血 AI 体验。
